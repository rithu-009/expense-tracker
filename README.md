# Expense Tracker

A command-line expense tracker built using Python and JSON storage.

# Features

- Add expenses to different categories
- View monthly expense summary
- View all expenses category-wise
- Edit existing expenses
- Delete expenses
- Search expenses by name
- Save data permanently using JSON
- Automatically load saved expenses on startup
- Input validation for dates and amounts

# Categories

- Groceries
- Fuel
- Snacks
- Phone Bill
- Medicine
- TV/Internet Bill
- Laundry/Ironing
- One-Time Expenses

# Technologies Used

- Python
- JSON
- File Handling
- Functions
- Dictionaries and Lists
- Exception Handling
- Datetime Module
